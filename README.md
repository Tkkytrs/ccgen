# ccgen
A cc generator py with many commands.
#commands

• --tkkytrs-cc-gen = generate cards.
• --tkkytrs-bin-chk = check bin.
• --tkkytrs-stripe-chkout = stripe checkout. (under devlopment)
• --tkkytrs-cc-live = check if card is live or not.
